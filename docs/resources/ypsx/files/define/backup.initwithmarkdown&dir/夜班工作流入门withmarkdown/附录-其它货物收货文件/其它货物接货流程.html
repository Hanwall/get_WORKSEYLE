#其它货物接货流程

  • 上面提到的蔬菜水产，猪肉和豆制品是每天必来的，除非大老板或者店长没下单那就不会来了

  • 这里提示其它偶尔会来的

##水果

大部分情况，水果白天来，但是偶尔会晚上来，没有单子，群内会通报来货，司机会说货量，搬货就像

##干货鸡蛋

大部分情况，鸡蛋白天来，但是偶尔会晚上来，搬货对单子就行

##快消馒头或者牛奶

馒头偶尔会来对单子就行

牛奶经常会来 不用搬货，司机会自己拿进来，跟运牛奶的司机一起对单就行，司机会自己陈列记得在单子上添加牛奶商品码

别的货白天来，不管
